Código: Descrição: Prioridade:
  Obra deve conter os seguintes atributos  Área(n), Tipo de Obra(1), Data de Inicio(1), Previsão de termino(1) e Data de Finalização(1).  Alta
  Toda Obra Cadastrada deve conter um Fornecedor responsável. Baixa
  Toda Obra Cadastrada deve conter pelo menos uma área  
  Toda Obra Cadastrada deve conter um Tipo de Obra, uma Data de inicio e uma previsão de termino.  
  Toda Obra finalizada deve ter a Data de Finalização  
  Cadastro, alteração e remoção de Obras só pode ser realizado por um usuário com nível de acesso Administrador  
  Visualização das Obras pode ser realizada por todos os usuários.  
  A Data de inicio, Tipo de Obra e Data de Finalização são parâmetros fixos
