Código: Descrição: Prioridade:
  Aviso deve conter os seguintes atributos Titulo(1), Descrição(1), Condomínio(1) e Data(1)  
  Cadastro, remoção e alteração de Avisos pode ser realizado por um usuário com acesso Administrador  
  Visualização dos Avisos pode ser realizada por todos os usuários.  
  Todo cadastro de Aviso deve conter o Titulo, Descrição, Condomínio e Data.   
  Na criação de avisos os Usuários de nível de acesso Pessoal recebem uma notificação via Software e via E-mail (Opcional)  
  O Aviso mais atual fica em destaque para os Usuários Operacionais
